~/.vimrc generator

~/.vimrc generator provides a simple method of generating a .vimrc configuration for vim.

The 'vimrc' file is a file that runs when vim is started. It is used to configure vim. Filling out the form below will generate a vimrc file with your specified configuration options. To use the vimrc file, simply save the generated file in your home directory as '.vimrc' (e.g. /home/user/.vimrc).
